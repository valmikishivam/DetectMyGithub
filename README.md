# DetectMyGitHub

A simple and powerful web application built with **HTML**, **CSS**, and **JavaScript**. Users can enter any GitHub username in the search bar, and the app will fetch and display their GitHub profile details with a direct link to visit their GitHub account.

## 🔗 Live Link

**View Live Website:** https://detectmygithub.netlify.app

## 🚀 Deployment Tech

*   **Platform:** Netlify

## 🛠️ Tech Stack

*   **HTML5:** Structure and semantic markup
*   **CSS3:** Styling, Flexbox, and responsive design
*   **JavaScript (ES6+):** DOM manipulation and GitHub API integration

## 🚀 Features

*   **Search GitHub Users:** Enter any username to find their GitHub profile
*   **Profile Display:** Shows avatar, name, bio, followers, following, and repositories
*   **Direct Link:** Button to visit the user's GitHub profile directly
*   **Responsive Design:** Works perfectly on mobile and desktop
*   **Error Handling:** Shows friendly error messages when user is not found
